# Changelog

All notable changes to Scribe will be documented in this file.

## [1.0.0] — 2026-03-21

### Added
- Initial release
- 3-column docs layout with responsive sidebar
- 6 content components: Callout, CodeBlock, CodeTabs, ApiEndpoint, Steps, Card
- Full-text search with keyboard navigation
- Dark/light mode with system detection
- Single-file configuration (`scribe.config.ts`)
- MDX support with pre-styled components
- shadcn/ui CSS variable compatibility
- Version switcher
- MIT license
